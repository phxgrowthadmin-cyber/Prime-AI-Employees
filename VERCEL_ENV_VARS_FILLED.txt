HOW TO ADD TO VERCEL
================================================================================

1. Go to: https://vercel.com/dashboard/nexus-ai
2. Click: Settings
3. Click: Environment Variables
4. For each variable:
   - Paste Key
   - Paste Value
   - If marked 🔒, check "Sensitive"

5. Click: "Add another" or just type next one
6. After all added, scroll down
7. Click: "Deploy" or "Redeploy"
